# 短视频剧本创意

Use this skill to turn a vague topic, product, account direction, or campaign goal into short-form video ideas and ready-to-shoot scripts. Default to Chinese output unless the user asks for English or a specific overseas platform.

## Source Note

This skill is an original synthesis of five high-relevance open-source short-video / creator-content / script-generation projects. For rankings and upstream links, read `references/top-sources.md` when the user asks about provenance, licenses, or why these methods were chosen.

## Core Workflow

1. Identify the goal:涨粉、转化、种草、教育、娱乐、品牌心智、直播引流、私域引流、账号人设建设。
2. Identify platform and audience: TikTok/Reels/Shorts/抖音/小红书/视频号/B站, audience pain, desire, objections, and viewing context.
3. Choose a script mode:口播、剧情、测评、清单、反差、挑战、访谈、教程、故事、UGC广告、AI视频分镜。
4. Generate 5-10 creative angles before writing the final script.
5. Pick or recommend the strongest angle using hook strength, novelty, emotional pull, clarity, and production feasibility.
6. Write the script with a 0-3s hook, beat-by-beat body, payoff, CTA, captions, shot notes, and edit rhythm.
7. Add variants for platform, length, tone, and audience maturity.
8. Include a quick self-check and iteration plan.

## Input Fields To Collect

Ask only for missing information that changes the output materially. If enough context exists, proceed with assumptions.

- 主题/产品/账号方向
- 目标平台和视频时长
- 目标用户是谁，TA最痛/最想要什么
- 视频目标:涨粉、成交、评论、收藏、转发、种草、留资
- 风格:专业、轻松、犀利、反差、搞笑、治愈、知识型、剧情化
- 是否真人出镜、口播、AI视频、图文混剪、产品实拍或无真人
- 禁区:不能夸大的功效、敏感词、品牌调性限制

## Creative Angle Bank

Use these angle types to create options before scripting:

- 痛点直击:把用户心里不敢说的问题说出来。
- 结果前置:先展示理想结果，再解释怎么做到。
- 反常识:挑战一个常见误区。
- 反差身份:看似普通的人说出高密度经验。
- 失败复盘:从踩坑切入，降低说教感。
- 清单收藏:3个方法、5个工具、7个避坑点。
- 对比实验:A方案 vs B方案，结果一眼可见。
- 故事转折:平静开头，突然出现冲突或反转。
- 低成本替代:不用昂贵方案，也能做到类似效果。
- UGC见证:普通用户第一视角，真实、具体、生活化。
- 热点嫁接:把热点话题转译到产品或账号领域。
- 评论区回应:用用户质疑作为开头。

## Hook Library

Write hooks that can be understood in under 2 seconds. Prefer one clear promise or conflict.

### Chinese Hook Patterns

- “别再[错误做法]了，真正有效的是[新方法]。”
- “如果你是[人群]，这条视频可能帮你少走[成本]弯路。”
- “我试了[数量]种方法，只有这个真的能[结果]。”
- “很多人以为[常识]，但实际问题出在[反常识点]。”
- “这是我第[数字]次劝你，不要在[场景]上浪费钱。”
- “看完这个，你会重新理解[主题]。”
- “一个[身份]不会告诉你的[秘密/流程/判断标准]。”
- “先别买[产品/服务]，除非你看懂这3点。”
- “这不是[表象]，这是[更深层原因]。”
- “评论区一直问的[问题]，今天一次讲清楚。”

### English Hook Patterns

- “Stop doing [common mistake]. Do this instead.”
- “If you are [audience], this will save you [cost/time/pain].”
- “I tested [number] ways to [goal]. Only one worked.”
- “Everyone thinks [myth], but the real reason is [insight].”
- “Before you buy [product], check these 3 things.”

## Script Structures

### 1. 口播知识脚本

Use for expertise, tutorials, opinion, personal brand, or high-save content.

```text
标题: [一句话结果/冲突]
时长: [30/45/60秒]
人设: [谁在说]
目标: [收藏/转发/评论/关注]

0-3秒 Hook: [一句强钩子]
3-8秒 共鸣: [用户正在经历的具体场景]
8-25秒 核心观点: [1-3个要点，每点配例子]
25-40秒 操作步骤/判断标准: [可执行]
40-50秒 反常识/补刀: [提升记忆点]
50-60秒 CTA: [评论关键词/收藏/关注/私信]

画面: [机位、字幕、B-roll]
字幕重点: [逐条列出]
```

### 2. 剧情短视频脚本

Use for短剧、情绪转折、品牌故事、生活化种草。

```text
人物: [A/B/旁白]
场景: [地点]
冲突: [一句话]
情绪曲线: [平静 -> 压迫 -> 反转 -> 释放]

0-3秒: 冲突前置，直接给观众一个问题。
3-10秒: 展示误会/压力/欲望。
10-25秒: 冲突升级，角色做出选择。
25-40秒: 反转或发现真相。
40-55秒: 情绪落点或产品自然出现。
55-60秒: 余味/金句/互动问题。

对白:
A: “...”
B: “...”
旁白: “...”

分镜: [景别、动作、道具、转场、音效]
```

### 3. UGC广告脚本

Use for产品种草、应用推广、电商、SaaS、课程、服务、线索转化。

```text
产品: [名称]
目标人群: [谁]
核心痛点: [什么问题]
核心卖点: [一个主卖点，最多三个辅卖点]
证据: [体验、数据、对比、前后变化、用户评价]

0-2秒 Hook: [痛点/结果/反常识]
2-8秒 场景: [用户真实生活里的麻烦]
8-20秒 发现: [产品如何进入场景]
20-40秒 演示: [具体怎么用，展示关键动作]
40-50秒 证据: [效果、对比、体验细节]
50-60秒 CTA: [点击、领取、评论、购买]

拍摄建议: 手机自拍感、自然光、真实语气、避免过度广告腔。
```

### 4. AI视频分镜脚本

Use when the script will become an AI video prompt or storyboard.

```text
主题: [一句话]
时长: [x秒]
比例: [9:16/16:9/1:1]
风格: [写实/电影/动画/国风/赛博朋克]

| 时间 | 画面 | 镜头 | 动作 | 字幕/旁白 | 音效 |
|---|---|---|---|---|---|
| 0-3秒 | ... | ... | ... | ... | ... |

可复制旁白: [完整口播]
可复制视频提示词: [自然语言分镜描述]
```

## Platform Adaptation

- 抖音/TikTok: hook更狠，前2秒给冲突或结果，节奏快，字幕短。
- 小红书:更像经验分享和种草笔记，强调真实体验、避坑、清单、审美。
- YouTube Shorts:开头要清楚主题，中段要有信息密度，结尾给订阅理由或系列化预告。
- Reels:视觉审美和生活方式感更重要，少解释，多展示。
- 视频号:可信度和中年/家庭/商业场景更重要，语气稳一点。
- B站短视频:观点和信息完整性更重要，可以加入梗但别牺牲逻辑。

## Quality Rules

- First line must make the viewer know why to keep watching.
- One video should have one main promise, one main conflict, or one main transformation.
- Avoid generic words like “干货满满”, “超实用”, “你一定要知道” unless followed by a concrete payoff.
- Prefer specific scenes:地铁上、下班后、直播间、厨房台面、老板会议前、孩子睡着后。
- Keep voiceover speakable. Read it aloud mentally; cut sentences that sound written, not spoken.
- Do not write illegal, deceptive, medical/financial guaranteed-result claims.
- UGC ads should feel like a person solving a real problem, not a brand reciting features.
- Every CTA should match the goal: comment for engagement, save for knowledge, click for conversion, follow for series.

## Default Delivery Style

Respond in crisp Chinese. If the request is broad, provide 5 creative angles and 1 fully written script. If the request is specific, lead with the finished script and add only the most useful shooting notes.
